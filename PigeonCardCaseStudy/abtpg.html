About Delta Insights

Introduction

Delta insights is a custom-made analysis tool designed to give directors and decision makers at Delta airlines an overview of the business performance. Data fetched from the database include a summary of deltaâ€™s fleet, routes and some data from the hangers. Furthermore, to ensure security, you will only be able to sign in and view the summarized cards on the applications dashboard if you possess valid login credentials which are provided and managed by the system administrator.

How to use it

In order to use this tool, all you have to do is to login with the provided username and password. Then after that, all you have to do is going through the card tiles which will provide a summary on specific items. Yes it is that easy!

Technology Used

This web application uses the nano-library pigeon-card to display the cards on the dashboard. This library is built using AngularJS and Bootstrap However, for the backend PHP is used to manipulate the database and to manage the login page
